# RAE-Suite: The Factory Orchestrator Manifest 🚀

## 🎯 Purpose
RAE-Suite is the **Unified Control Plane and Ecosystem Wrapper** for the entire Autonomous AI Factory. It provides the glue, the network, and the governance necessary for all modules to function as a single, coherent organism.

## 🧠 Core Pillars
1.  **Unified Network (rae-internal)**: Establishes the secure neural network connecting Memory, Architect, Execution, Auditor, and Observatory.
2.  **Shared DNA (rae-core Integration)**: Ensures that every module shares the same underlying contracts, models, and Enterprise Guard protocols.
3.  **Global Factory State**: Orchestrates the startup, health, and scaling of all services through standardized Docker configurations.
4.  **Strategic Integration**: Manages the relationship between the Factory (RAE) and its Products (ScreenWatcher, Billboard Marker).

## 📊 Strategic KPIs
*   **Ecosystem Uptime**: Availability of all core services.
*   **Integration Density**: Depth of communication between Phoenix, Hive, and Quality.
*   **Ecosystem Security Score**: Global audit result based on all module outputs.
