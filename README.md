# RAE-Suite v4.3: Global AI Engineering Factory 🚀

RAE-Suite is an integrated orchestration environment for intelligent engineering agents. The system operates as an "Autonomous Factory" that designs, develops, tests, and monitors Enterprise-grade software.

## 🏗️ Factory Architecture (Quantum Ready)

The system consists of 5 specialized modules connected into a single neural network:

1.  **[RAE-agentic-memory](packages/rae-core)**: The Quantum Intelligence Core. Manages semantic knowledge, ontology, and advanced reflection.
2.  **[RAE-Phoenix](packages/rae-phoenix)**: Lead Architect and Developer. Generates code, manages refactoring, and performs deep repository analysis.
3.  **[RAE-Hive](packages/rae-hive)**: Execution Layer (The Hands). Runs scripts, manages Docker infrastructure, and performs visual validation via Playwright.
4.  **[RAE-Quality](packages/rae-quality)**: Autonomous Quality Guard. Performs continuous security auditing (SAST), vulnerability scanning, and ensures high test coverage.
5.  **[RAE-Lab](packages/rae-lab)**: Intelligence Observatory. Aggregates metrics from all agents to drive Kaizen (Continuous Improvement) and Lean strategies.

## 🛡️ Standards & Security
*   **ISO 27001 Compliance**: Full separation of duties and total auditability of agent actions.
*   **Hard Frames 2.1**: Contractual enforcement to ensure reliable and non-hallucinatory AI outputs.
*   **Telemetria & Observability**: Real-time operational data integrated with Grafana.

## 🚀 Quick Start
```bash
# Clone with submodules
git clone --recursive git@github.com:dreamsoft-pro/RAE-Suite.git

# Launch the entire Factory
docker compose -f docker-compose.suite.yml up -d
```
